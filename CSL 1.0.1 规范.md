# CSL 1.0.1 Specification

## 介绍

CSL 是一种基于 XML 的格式，用来描述引用的格式，注释和参考文献，提供了

- 一种开放的格式

- 紧凑并且稳定的格式

- 对格式要求的外部支持

- 格式的发布和更新的基本支持

- 数千种免费提供 的样式

  

有关的其他文档，CSL 介绍，样式和 locale 详见 [CSL 项目主页](https://citationstyles.org/)
